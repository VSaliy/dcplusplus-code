Favorite users

Columns

Auto grant slot / Nick
Specifies the nick of the user. The checkbox indicates if DC++ should auto grant the user a slot. If the box is checked, DC++ will automatically grant the user a slot.
Hub (last seen in, if offline)
Which hub you saw this user the last time.
Time last seen
Which specific date and time you saw this user the last time; or "Online" if the user is currently online.
Description
Brief description of the user. You can change this description by either right-clicking and selecting "Description", or by pressing the Enter key.
CID
Client ID of the user.

Context menu commands

Connect to hub
Connect to the hub where the user last seen in.
Description
Edit the brief description of the user (shown in the Description column).

Description of some other common context menu options are listed in the General Menu Commands page.
