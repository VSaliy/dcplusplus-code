Notifications

Download finished
This event fires every time a download completes. Doesn't apply to file lists.
File list downloaded
This event fires every time a file list download completes.
Main chat message received
This event fires every time a chat message is received in the main channel of a Hub.
Private message received
This event fires every time a private chat message is received.
Private message window opened
This event fires every time the reception of a private chat message leads to a new Private Message window being opened.

Play a sound

Checking this box will turn sound notifications on for the selected event. The system's default beep sound will be used by default unless a custom one is assigned from the "Notification sound" group.
This box can be switched by double-clicking the sound icon in the "Sound" column of the desired event.

Display a balloon popup

Checking this box will turn balloon popup notifications on for the selected event. An example of such a balloon can be previewed by pressing the "Fire a balloon popup example" button.
This box can be switched by double-clicking the balloon icon in the "Balloon" column of the desired event.

Notification sound

Instead of the system's default beep, a custom sound can be assigned to each event by selecting the desired event in the "Notifications" list, enabling the "Sound" icon, then clicking the "Browse..." button. Only Wave sound files (typically with a ".wav" extension) are allowed.

Fire a balloon popup example

Pressing this button will have DC++ launch a balloon popup example from the Windows notification area. The balloon popup will be titled "DC++ - Balloon popup example" and will contain the following string as its content text: "This is an example of a balloon popup notification!".
