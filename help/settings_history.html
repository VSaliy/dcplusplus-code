History

Chat lines to recall from history when opening a window

Hubs
The number of lines that will be read from the main chat log and displayed when a new Hub window is opened. (default: 10)
Note that main chat logging must be enabled.
Private messages
The number of lines that will be read from the private message log and displayed when a new Private Message window is opened. (default: 10)
Note that private message logging must be enabled.

Maximum number of recent items to save

Recent hubs
The maximum number of recent Hub windows that DC++ keeps track of.
Recent PMs
The maximum number of recent Private Message windows that DC++ keeps track of.
Recent file lists
The maximum number of recent File List windows that DC++ keeps track of.

Search history

The number of old search lines that will be shown in the Search drop-down controls of the Search and File List windows. (default: 10)
