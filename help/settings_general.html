Personal information

This is general information about yourself. The nick is required, though it's suggested that you pick the correct upload line speed value as well.

Nick
This nickname, or handle, will identify you when you join a hub. It can be overridden on a per-hub basis in the Favorite Hub Properties.
E-Mail
This field is for sharing your email address, if you wish it.
Description
This description will show in the user list. It can be overridden on a per-hub basis in the Favorite Hub Properties.
Line speed (upload)
This drop-down contains a number of common upload speeds (in MiB/s), please choose the one closest to your own. Do not put in your download speed; this setting is a measure of how fast a source you are likely to be for other users. This value will be shown for the users on NMDC hubs only.

Away mode

These settings control the away mode. It is a mode that DC++ can be switched to to signify that you are away from DC++ and unavailable at the time. You can activate away mode by using the Away indicator in the status bar of the main window, the /away <message> and /back chat commands, the Auto-away on minimize option or the Auto-away when Windows is locked option.

Default away message
This is the default away message that will appear when someone PMs you for the first time while you are in the away mode. All time formatting variables are available, see Logs. Use Ctrl + Enter to create a new line.
(default: "I'm away. State your business and I might answer later if you're lucky.")
Auto-away on minimize (and back on restore)
Switch to away mode whenever the client is minimized in the Windows taskbar or in the notification area (and set away mode off whenever restored).
Auto-away when Windows is locked (and back when unlocked)
Switch to away mode whenever the Windows session is locked (generally by Windows key + L or via Ctrl + Alt + Del), and set away mode off when the session is unlocked.
