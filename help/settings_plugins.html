Plugin configuration

This page is for individual configuration, addition and removal of plugins made for DC++. Note that the plugin system is in experimental state and the format of the loadable plugins may change in the future.

Installed plugins

List of currently installed plugins.

Add
Open a file selection dialog and add the selected plugin to the list if the plugin format is determined correct. Some plugins may require you to restart DC++.
Configure
Open the plugin's own configuration page in order to edit its properties.
Move Up
Move the currently selected plugins upwards in the list.
Move Down
Move the currently selected plugins downwards in the list.
Remove
Stop and remove the currently selected plugins.

Plugin information

This panel shows all the available information from the selected plugin.
