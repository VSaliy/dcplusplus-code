Plugin configuration

This page is for individual configuration, addition and removal of plugins made for DC++.

Installed plugins

List of currently installed plugins.

Add
Open a file selection dialog and offer to install the selected plugin if it is compatible with DC++.
Some plugins may require you to restart DC++.
Configure
Open the selected plugin's own configuration page in order to edit its properties.
Enable
Enable the selected plugin.
Disable
Disable the selected plugin.
Move Up
Move the selected plugin upwards in the list.
Move Down
Move the selected plugin downwards in the list.
Remove
Disable and uninstall the selected plugin.

Plugin information

This panel shows information about the selected plugin.
