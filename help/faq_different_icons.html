What do the different user icons mean?

Base icons:
Green
Standard user icon (no particular attribute).
Grey
The user is in away mode.
Robot
The user represents a bot the hub uses to issue automatic commands.
Icon modifiers:
Yellow key
The user is a hub operator.
Red bricks
The user cannot be connected to, for he is in passive mode and does not support NAT traversal.
Red circle
The user has no free slot available.
