Toolbar customization

You can use a standard Windows toolbar customizaton dialog box to customize (add, remove, and rearrange) the toolbar items in the main DC++ window.

Step by step instructions for using the toolbar customization dialog box

  1. To add a button or a separator to the toolbar, select one from the "Available buttons" list box and click "Add ->".
  2. To remove a button or a separator from the toolbar, select it from the "Toolbar buttons" list box and click "<- Remove".
  3. To reposition a button on the toolbar, select it from the "Toolbar buttons" list box and click either "Move Up" or "Move Down" to move the button accordingly.
  4. To return the toolbar to the state it was in when the dialog box was opened (undo changes), click the Reset button.
  5. When your changes are complete, click the Close button to save your changes.
