Connections

This lists your current connections to other peers in the network, uploads (from you to the other peer) and downloads (from the other peer to you).

Columns

User
The nick of the connected user
Hub
Displays the hub which the user connected through
Status
Shows the current status of the connection which includes Connecting, Downloading, Uploading, No slots available, Idle, Disconnected or Terminated. During data transfer you can see here the name of the file, the current chunk of the file being downloaded from the connected peer and maybe one or more letters in [box brackets] as well. These letters specify the properties of the transfer:
  • [T] - TTH Checked
  • [Z] - ZLib Compressed transfer
  • [S] - Trusted secure connection
  • [U] - Untrusted secure connection
Speed
The current up or download speed
Chunk
Remaining and total size of the current chunk being downloaded
Transferred (Ratio)
Shows the total number of data bytes transferred since the connection was established. The compression ratio (in case of compressed transfers) is added between parentheses
Queued
During download this column displays the total size of files in the queue available from this connection
Cipher
In case of a secure connection it shows the encoding method
IP
IP address and country code (if available) of the connected peer

Context menu commands

Force attempt
Forces DC++ to try to connect to the user immediately.
Disconnect
Forces DC++ to disconnect the user immediately.

Description of some other common context menu options are listed in the User-related menu commands page.
