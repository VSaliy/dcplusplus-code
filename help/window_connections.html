Connections

This lists your current connections to other peers in the network, uploads (from you to the other peer) and downloads (from the other peer to you).

Columns

User
The nick of the connected user
Hub
Displays the hub which the user connected through
Status
Shows the current status of the connection which includes Connecting, Downloading, Uploading, No slots available, Idle, Disconnected or Terminated. During data transfer you can see here the name of the file, the actual chunk of the file beeing downloaded from the connected peer and may one or more letters in [box brackets] as well. These letters specify the properties of the transfer :
  • [T] - TTH Checked
  • [Z] - ZLib Compressed transfer
  • [S] - Trusted secure connection
  • [U] - Untrusted secure connection
Speed
The actual up or download speed
Chunk
Remaining and total size of the actual chunk beeing downloaded
Transferred (Ratio)
Shows the total number of data bytes transferred since the connection's estabilished. The compression ratio (in case of compressed transfers) added in parentheses
Queued
During download this column displays the total size of files in the queue available from this connection
Cipher
In case of a secure connection it shows the encoding method
IP
IP address and country code (if available) of the connected peer
