Finished uploads

Note that file list logging must be enabled for them to appear in this window.

The list of finished files are grouped in two ways: by files and by users. You can select between the two grouped lists using tabs that appear in the top of the window. In both lists a new item will appear right away when the first chunk of a file uploaded.

Columns

Time
The date and time the file finished uploading.
Filename
The name of the uploaded file.
Path
The path to the uploaded file.
Nicks
To whom the file was uploaded.
File size
The size of the file uploaded.
Transferred
The number of transferred bytes.
% transferred
Shows how many percent of the file transferred (this value can exceed 100% if the file was uploaded to more than one user).
Speed
The average speed for the file.
CRC Checked
Whether or not the file was CRC checked. Note that Enable automatic SFV checking must be enabled, and the file needs to have an accompanying SFV file.
Hub
Which hub the user was on.
Files
Which files were uploaded to the user.
Elapsed
The total time, in seconds, that it took for the file to be transferred.

Context menu commands

View as text

It will open the file (stored locally) in a text viewer window.

Status bar

# Items
The amount of items that you have uploaded (sent).
# bytes
The amount in bytes that you have uploaded (sent).
# bytes/seconds
The average speed of the files you have uploaded (sent).
