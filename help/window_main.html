Main DC++ window

Main Menu bar (top)

File Menu
You can find the file input-output functions as well as the basic connection options and the settings in this menu.
Quick Connect...
Allows you to connect to a hub by manually typing its address.
Reconnect
To reconnect (disconnect and connect again) the hub which is currently in the foreground. If the active window is not a hub, it does nothing.
Follow last redirect
You are able to follow the redirection offered by the hub which is currently in the foreground. If the active window is not a hub, it does nothing.
Open file list...
This menu is for opening a previously downloaded filelist. It defaults to the FileLists directory in your DC++ program folder.
Open own list
You can open and check how your own file list looks like. Useful to see what's displayed to others when they get your file list.
Match downloaded lists
This function is used to match your download queue for already downloaded file lists (they stored in the FileLists directory in your DC++ program folder).
Refresh file list
This will immediately rescan your shared directories for new or changed content and updates your file list.
Open downloads directory
Opens your downloads folder in Windows Explorer.
Settings...
Opens the Settings panel.
Exit
Closes DC++.
View Menu
You can open all DC++ tabs and windows with the sub-items of this menu. Some kind of tabbed windows can have only one instance opened while you are able to open more from others. If you try to open more from a tab which can't have more than one instance, the already opened one will be focused.
Public Hubs
Opens the Public Hubs window. You can retreive lists of available hubs and connect to them here.
Favorite Hubs
Opens the Favorite Hubs window where you can store and manage you favorite places in the DC network as well as set to autoconnect any of them when you start DC++.
Favorite Users
Opens the Favorite Users window. You can store and manage your favorite DC users here as well as grant extra access to your share for any of them.
Download Queue
Opens the Download Queue window. You can track and manage your files beeing downloaded here.
Finished Downloads
Opens the Finished Downloads window which lists your downloaded files in the current session. Right clicking the list items you are able to do anything with the (finished) files, just as in Windows Explorer.
Waiting Users
Opens the Waiting Users window. It lists those users who are currently waiting for a free slot to be able to start downloading from you.
Finished Uploads
Opens the Finished Uploads window which lists the files uploaded from you in the current session. Right clicking the list items you are able to do anything with the files, just as in Windows Explorer.
Search
Open the Search window. This is the place where you can search for files to download.
ADL Search
Opens the ADL Search window. It incorporates a tool for filtering file lists downloaded from users.
Search Spy
Opens the Search Spy window where you are able to spy other users' searches.
Notepad
Opens the Notepad window, where you can store random text. The text will be kept even if you close DC++.
System Log
Opens the System Log window, where all system messages of the current session are logged.
Network Statistics
Opens the Network Statistics window, which displays a real-time chart of the avarage speed of data sent and received by DC++.
Indexing progress
Opens the Indexing progess dialog, which shows you the current state of file hashing operation and how much time there is left from it.
Window Menu
You can find options here to close all instance of specific groups of tabs by one click.
Close disconnected
Closes all disconnected Hub windows.
Close all PM windows
Closes all private message windows regardless of their online status.
Close all offline PM windows
Closes all offline private message windows.
Close all file list windows
Closes all opened file list windows. Saved file lists can be reopened by selecting Open File List... from the File menu.
Close all search windows
Closes all opened search windows.
Help Menu
Here you can access the built-in Help, FAQs, guides and some documentation. It also contains hotlinks to reach the different parts of the DC++ online community.
Help Contents
Opens the Overview page of the built-in Help.
Get started
The best way to start here if you are new to DC++.
Change Log
You can follow what's been changed in the different DC++ versions since the development started.
About DC++...
You can see the copyrights, list of all names who ever contributed to DC++ as well as some useful statistical information about your up/download ratio. The About box also shows you the version number of the current stable release of DC++.
DC++ Homepage
Opens the DC++ home page in your web browser.
Downloads
Opens the trusted DC++ download page in your web browser.
GeoIP database update
Opens a web page where the latest GeoIP database can be obtained. The database file called GeoIPCountryWhoIs.csv and should be saved to the DC++ program folder. GeoIP is used to show what country uploaders and downloaders are from in the Connections window. The database file is frequently updated on the web page - the newer your GeoIP file is the more exact country information will be displayed in the Connections window.
Frequently asked questions
Opens the official Direct Connect FAQ page in your web browser.
Help forum
Points to the DC++ support forum.
DC++ discussion forum
Points to the DC++ discussion forum.
Request a feature
You can propose modifications or request new features here.
Report a bug
You can report problems (only) about DC++ here.
Donate (paypal)
Found this free program useful? Then you may consider donate the further development... You can do it here, through the author's Paypal account.

Tabs

Each window opened in DC++ has a corresponding tab, which appears in the global tab control.

Tabs in DC++ behave the same as in standard Windows applications; moreover, they have the following additional features:

  • They can be moved around in a drag-and-drop fashion.
  • There are some easy ways to switch between them: with mouse extra buttons or using Ctrl+Tab.
  • When many tabs are opened, and they can't fit into a single row, they will automatically occupy more rows.
  • Each tab has its own context-menu which provides commands related to the window the tab belongs to. Description of the common context menu options are listed in the General Menu Commands page.
  • Tabs can be closed with:
    • Middle mouse click
    • Shift + left mouse click
    • The "Close" command from the tab context-menu
    • Ctrl+W
    • Ctrl+F4
You can customize the look and behavior of tabs from the Tabs settings and Windows settings respectively.

Status bar (bottom)

The main status bar is always visible in the bottom of the DC++ window. You can get information from the current state of several parameters of DC++ here. The status bar is divided into sections for better visibility. Their meaning from left to right:
System messages
The first part of the status bar contains the last system message with a timestamp. To see more from the recent system messages you can move the mouse pointer above the statusbar - the last 10 lines of the messages will be shown as a tooltip shortly. To see all system messages of the current session open the System Log window from the View menu or simply double-click on this part of the status bar.
Away indicator
Shows "AWAY" when DC++ is in away mode.
Double-click to switch away mode on and off.
Hubs
Shows the number of hubs you are currently connected to in the following format: x/y/z, where:

x = number of connected hubs where you're not a registered user;
y = number of connected hubs where you're registered in;
z = number of connected hubs where you're an operator;

just the same way as in the DC++ tags of the users listed on a hub.
Slots
Indicates the number of currently free upload slots out of the total number of slots opened.
Slots up-down control
Use this control, which consists of 2 arrows, to increase (up arrow) or decrease (down arrow) the number of upload slots you want to have opened.
Total downloaded bytes
Shows the total number of bytes received in the current session, including file transfers, hub-client communication and hublists.
Double-click to open the Network Statistics window.
Total uploaded bytes
Shows the total number of bytes sent in the current session, including file transfers and hub-client communication.
Double-click to open the Network Statistics window.
Current download speed
Shows the current overall download speed and the number of current download connections in parentheses (updated in every second).
Double-click to open the Network Statistics window.
Current upload speed
Shows the current overall upload speed and the number of current upload connections in parentheses (updated in every second).
Double-click to open the Network Statistics window.
