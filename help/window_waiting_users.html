Waiting users

This page will display those users who have queued a file from you, but haven't been able to get a slot. As well, from which hub they originate, the file that they've queued and how much of the file they have completed.

Note that this is not an upload queue, and just a visual representation of who have tried to get a slot for what.

Menu options

Get file list
Get the file list of the selected user.
Copy filename
Copy the file name of the queued file.
Remove
Remove a user from the list.
Grant extra slot
Grant the user an extra slot, beyond your set slots.
Add to favorites
Add the user to your favorite users.
Send private message
Send the user a private message.
